# 🎓 Symbiosis Admission Form Validation & Automation

This project is a simple web-based admission form with validation using JavaScript and automated testing using Selenium (Python).

---

## 📌 Project Overview

The project demonstrates:
- Frontend form design using HTML & CSS
- Client-side validation using JavaScript
- Automated form testing using Selenium WebDriver

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Selenium WebDriver

## ✨ Features

- User input fields:
  - Name
  - Email
  - Password
  - Confirm Password
  - Gender
  - Course selection

- Validation checks:
  - Empty fields check
  - Password length validation
  - Password match validation
  - Course selection validation

- Alerts for user feedback

- Automated testing using Selenium:
  - Auto-fills form data
  - Selects options
  - Submits form
